# CRM для СНТ (Садоводческого Некоммерческого Товарищества)

Система управления садоводческим товариществом: учёт участков и владельцев, начисление взносов, приём платежей, импорт банковских выписок, телефония.

## 🚀 Возможности

- **Управление участками** — кадастровые номера, площадь, координаты, владельцы
- **Реестр владельцев** — контакты, история владения, членство в СНТ
- **Начисления и платежи** — членские/целевые взносы, электричество, пени
- **QR-квитанции** — генерация по ГОСТ Р 56042-2014 с реквизитами СНТ
- **Импорт банковских выписок** — 1С, Сбербанк, ВТБ (Excel/PDF)
- **Автосопоставление платежей** — по UID, ФИО, участку
- **Массовые рассылки** — email-квитанции должникам
- **Телефония** — интеграция с Asterisk (AMI)
- **Суб-аккаунты** — роли: администратор, председатель, бухгалтер, наблюдатель
- **Мульти-СНТ** — поддержка нескольких организаций

## 📦 Технологический стек

- **Backend**: Python 3.13, Django 6.0, Django REST Framework
- **Database**: PostgreSQL
- **Server**: Gunicorn + Nginx
- **Frontend**: Vanilla JS, HTML/CSS
- **Телефония**: Asterisk AMI
- **Генерация PDF**: WeasyPrint, ReportLab
- **QR-коды**: qrcode (ГОСТ Р 56042-2014)
